# PCA-and-KNN
PCA降维与KNN分类可视化
1. 主成分分析（PCA）
PCA 是一种经典的无监督线性降维方法。它通过正交变换将原始数据投影到新的坐标系上，使数据在新坐标系上的方差最大。新坐标轴被称为主成分。
● 核心目标： 找到 k 个正交方向，使数据投影到这些方向上时，丢失的信息最少（即保留的方差最大）。
● 方差贡献率： 每个主成分的方差贡献率表示该主成分捕获了原始数据中多少信息。

2. K 近邻算法（KNN）
KNN 是一种有监督的懒惰学习算法，它依赖于距离度量。给定一个测试样本，KNN 算法在训练集中找到与该样本距离最近的 K 个邻居，然后根据这 K 个邻居的类别进行投票，决定测试样本的类别。常用的距离度量包括欧式距离。
● 降维的意义： 当原始数据维度过高时，距离计算会变得不可靠（维度灾难）。PCA 可消除冗余特征，使数据在低维空间中更好地分离，可能提升 KNN 性能。
